# Breakout

Original Atari Breakout recreated in C++.

You must have [SplashKit](https://splashkit.io/) installed.

First setup the build:
```bash
mkdir build
cmake -S . -B build
```

To build this run:
```bash
cmake --build build
```

Then to run it:
```bash
build/breakout
```

Or if on Windows:
```bash
build\breakout.exe
```
